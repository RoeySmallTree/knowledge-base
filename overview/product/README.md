## Overview

Cabal's product is built around these core concepts:

1. **Workspaces** - Organization/account containers
2. **Sessions** - Individual project/mission containers
3. **Missions** - Clear objectives derived from user prompts
4. **Agents** - AI roles that collaborate (Chair, Operatives, Watchdog, Envoy)
5. **Product Tree** - Structured outputs (content, collections, decisions)
6. **Rounds** - Collaboration lifecycle phases
