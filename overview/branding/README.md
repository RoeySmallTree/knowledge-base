# Branding - Context Guide

## Purpose

This area defines CABAL's brand identity including tone, voice, personality, visual identity, and messaging guidelines. Use these files to ensure consistency across all communication and user-facing materials.

## Quick Navigation

- **Start here:** [vibe.md](./vibe.md) - Brand tone, voice, and personality
- **For visual guidelines:** [visual-identity.md](./visual-identity.md) - Colors, aesthetics, design principles
- **For messaging:** [messaging.md](./messaging.md) - Key messages, taglines, and language patterns

## File Descriptions

### vibe.md
**When:** Writing any user-facing content or communication
**What:** Brand personality, tone, voice, and how we express ourselves

Use when you need to ensure content sounds like CABAL and maintains brand consistency.

### visual-identity.md
**When:** Creating visual materials, UI/UX, or discussing design
**What:** Visual brand elements including colors, typography, design principles

Use when making design decisions or describing visual aspects of the product.

### messaging.md
**When:** Creating marketing materials, taglines, or key messages
**What:** Core messages, taglines, and language patterns that define how we talk about CABAL

Use when crafting headlines, taglines, or key marketing copy.

## When to Explore This

**Explore when:**
- Writing user-facing content (UI copy, documentation, marketing)
- Creating visual materials or design elements
- Reviewing content for brand consistency
- Onboarding new contributors who will create content
- Making decisions about product personality

**Specifically read vibe.md when:**
- Writing any copy or content
- Unsure about tone or voice
- Need to maintain brand personality

**Specifically read visual-identity.md when:**
- Making UI/UX decisions
- Creating graphics or visual materials
- Discussing design direction

**Skip when:**
- Working on internal backend code with no user-facing elements
- No content creation or design decisions needed
- Already familiar with brand from current session
