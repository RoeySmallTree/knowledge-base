# Pitching - Context Guide

## Purpose

This area contains pitch materials and messaging for different audiences and contexts. Use these files when communicating CABAL's value proposition to stakeholders, users, investors, or partners.

## Quick Navigation

- **Start here:** [elevator-pitch.md](./elevator-pitch.md) - 30-second pitch for quick introductions
- **Main pitch:** [one-pager.md](./one-pager.md) - Canonical pitch text
- **For audience context:** [target-audience.md](./target-audience.md) - Who we're pitching to and their concerns

## File Descriptions

### elevator-pitch.md
**When:** Quick introductions, networking, initial conversations
**What:** 30-second pitch that captures CABAL's essence

Use when you need to quickly communicate what CABAL is and why it matters.

### one-pager.md
**When:** More formal pitches, meetings, written proposals
**What:** Comprehensive pitch covering problem, solution, approach, and value

Use when you have more time and need to provide detailed information.

### target-audience.md
**When:** Tailoring messaging, understanding stakeholder needs
**What:** Definitions of different audience segments and what matters to them

Use when customizing pitch for specific audiences or understanding user concerns.

## When to Explore This

**Explore when:**
- Writing marketing or promotional copy
- Preparing for stakeholder presentations
- Creating user-facing content
- Need to explain CABAL to someone new
- Drafting grant proposals or partnership pitches

**Skip when:**
- Working on internal technical implementation
- No need to communicate with external parties
- Already familiar with pitch from current session
