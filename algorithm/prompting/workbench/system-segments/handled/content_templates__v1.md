### CONTENT TEMPLATES BY TYPE

**Content Product:**
```markdown

{Why this exists, what problem it solves}

**Decision Product:**
```markdown
# {Decision Question}

## Context
{Background and constraints}

## Options
1. **Option A** — {description}
   - Pros: ...
   - Cons: ...
2. **Option B** — ...

## Recommendation
{Chosen option}

## Rationale
{Why this choice}

## Implications
{What this means for dependent work}
```
