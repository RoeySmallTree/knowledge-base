# The Personal University

> *"Course that adapts to your learning gaps proactively."*

## Overview

The Personal University is an adaptive learning system that builds personalized curricula. Unlike static courses, it anticipates confusion, adjusts to learning pace, and builds prerequisite trees dynamically.

## Tags
`#cortex` `#learning` `#education` `#personalized` `#adaptive`

## Use Cases
- Self-directed learning
- Professional development
- Skill acquisition
- Knowledge gap filling
- Test preparation

## Composition

| Role | Agent | Function |
|------|-------|----------|
| Chair | The Dean | Curriculum designer—owns learning path and progression |
| Operative | Professor | Content delivery—lectures, explanations, demonstrations |
| Operative | TA | Practice and assessment—quizzes, exercises, feedback |
| Operative | Devil's Advocate | Anticipates confusion—pre-creates explanations for common mistakes |

## Expected Outputs
1. **Syllabus Tree** — Visual learning path
2. **Learning Modules** — Content organized by topic
3. **Quizzes** — Assessment at each stage
4. **Anticipated FAQ** — Pre-created confusion resources
5. **Progress Dashboard** — Tracking and recommendations
